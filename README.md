# CRUD em PHP com MySQL

Projeto simples de estudo para aprender CRUD usando PHP e MySQL.

## O que é CRUD?

CRUD significa:
- Create (Criar)
- Read (Ler/Listar)
- Update (Editar)
- Delete (Apagar)

## Objetivo do projeto

Este projeto foi criado para aprender na prática:
- PHP
- MySQL
- Conexão com banco de dados
- Uso básico de Git e GitHub
- Boas práticas básicas de organização e segurança

## Funcionalidades

- Cadastrar usuários
- Listar usuários
- Editar usuários
- Excluir usuários
- Exibir mensagens de sucesso ou erro

## Tecnologias utilizadas

- PHP
- MySQL
- HTML
- XAMPP
- Git e GitHub

## Segurança do projeto

Foi utilizado o arquivo `.gitignore` para evitar o versionamento de arquivos sensíveis,
como o arquivo de conexão com o banco de dados, ajudando a manter informações seguras.

## Estrutura do projeto

crud-php/
index.php
criar.php
editar.php
processa.php
processa_editar.php
processa_deletar.php
conexao.php
README.md

## Banco de Dados

Nome do banco: PrimeiroCRUD  
Tabela: Usuarios

Campos da tabela:
- id
- nome
- email
- criado

## Como rodar o projeto

1. Instale o XAMPP
2. Clone este repositório
3. Coloque o projeto na pasta htdocs
4. Crie o banco de dados no phpMyAdmin
5. Acesse o projeto pelo navegador

## Observação

Este projeto é apenas para estudo e aprendizado.  
Não possui foco em visual.
